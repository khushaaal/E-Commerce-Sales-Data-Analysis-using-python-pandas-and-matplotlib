# 📊 E-Commerce Sales Data Analysis

An advanced E-Commerce Sales Analysis project built using Python, pandas, and Matplotlib.

This project analyzes sales data, calculates revenue statistics, identifies top-selling products, and generates visual charts for business insights.

---

# 🚀 Features

- Load and analyze CSV sales data
- Data cleaning and preprocessing
- Remove duplicate and missing records
- Calculate:
  - Total Revenue
  - Average Order Value
  - Highest & Lowest Sales
- Identify top-selling products
- Category-wise revenue analysis
- Most profitable product detection
- Generate sales reports automatically
- Create:
  - Bar Chart
  - Pie Chart
  - Line Graph

---
